## St Andrews University Students in the Colonies 

<img src="colonies.jpg"/>

This is a [`Svelte`] project. It includes a deployment setup using [`gh-pages`] to publish the built site to GitHub Pages.

### Prerequisites

- [`Node.js`] 
- [`npm`] installed globally

### Installation

npm install

### Development

npm run dev 

### Build

npm run build

### Deploy

npm run deploy
